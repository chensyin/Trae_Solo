# 氢能项目维度搜索

## 概述

项目维度负责采集氢能项目全生命周期信息，包括：
- 项目前期（备案/环评/招标）
- 项目后期（开工/建设/投产/验收）
- 按项目类型遍历

## 项目维度评分标准

| 评分 | 标准 | 来源类型 |
|------|------|---------|
| **5分** | 政府审批平台原文（备案/环评/招标/中标公示），含项目名称、投资规模、建设内容 | 发改委/生态环境厅/招标平台 |
| **4分** | 企业/政府发布的开工/投产新闻，含具体投资额、产能规模、参与企业 | 企业官网/政府新闻/行业媒体 |
| **3分** | 一般媒体的项目报道，含基本项目信息 | 综合媒体 |
| **2分** | 内容泛泛、缺乏项目具体信息，或非当月 | 自媒体/论坛 |
| **1分** | 无关内容 | 广告 |

## 项目维度"充分"标准

| 检查项 | 充分标准 |
|--------|---------|
| 前期项目（备案/环评/招标） | ≥5条（反映未来项目储备） |
| 建设期项目（开工/建设） | ≥3条（反映在建项目进展） |
| 投产项目（竣工/投产/投运） | ≥3条（反映产能释放） |
| 测试/验收项目 | ≥2条（反映技术验证进展） |
| 项目类型覆盖 | 至少覆盖4种不同项目类型 |
| 区域覆盖 | 至少覆盖5个不同省份 |

## 与产业维度的边界划分

| 信息类型 | 归属维度 | 判断标准 |
|---------|---------|---------|
| 企业A中标企业B的电解槽订单 | **产业** | 关注点是企业间的商业关系 |
| 某绿氢项目EPC招标 | **项目** | 关注点是具体项目的建设进展 |
| 企业获得融资 | **产业** | 关注点是资本动态 |
| 某项目开工/投产 | **项目** | 关注点是项目里程碑 |
| 电解槽价格走势 | **产业** | 关注点是市场数据 |
| 某项目投资规模/产能 | **项目** | 关注点是项目参数 |
| 某项目设备安装/调试 | **项目** | 关注点是项目进展 |
| 某企业发布新产品 | **产业** | 关注点是产品/技术 |

**简单判断**：以**具体项目**为中心（有项目名称、所在地、投资规模）→ 项目维度；以**企业/市场**为中心（有企业名、金额、市场趋势）→ 产业维度。

## 弹性搜索（按需触发）

> **触发条件**：当基础搜索（J1-J34, J10e-J10j, J19a-J19d）完成后，覆盖度评估显示"不足"时，按以下优先级补充搜索。每轮补充1-3次，最多3轮。连续3轮无新增≥3分结果则标记"已搜尽"。

### 补充搜索优先级

| 优先级 | 补充方向 | 搜索语句模板 |
|--------|---------|-------------|
| 1 | 按热点深挖 | `"[热点关键词]" (备案 OR 环评 OR 招标 OR 投产) [月份] [年份]` |
| 2 | 按未覆盖区域补充 | `"氢能" 项目 [未搜索省份] [月份] [年份]` |
| 3 | 按未覆盖项目类型补充 | `"[未覆盖项目类型]" (备案 OR 环评 OR 招标 OR 投产) [月份] [年份]` |
| 4 | 按央企项目补充 | `"[央企名]" 氢能 项目 [月份] [年份]` |
| 5 | 换词重搜 | 将"备案"→"立项"/"核准"/"审批"等同义词替换后重搜 |

### 换词原则

- 更换同义词：`备案`→`立项`→`核准`→`审批`→`签约`
- 更换区域：`内蒙古`→`新疆`→`甘肃`→`宁夏`→`青海`→`西藏`
- 更换项目类型：`风光制氢`→`光伏制氢`→`输氢管道`→`SAF`→`液氢`
