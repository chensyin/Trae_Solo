# 氢能政策维度搜索

## 概述

政策维度负责采集氢能相关的政策信息，包括：
- 中央部委政策（国家能源局、工信部、发改委、财政部等）
- 地方政府政策（各省市发改委、工信厅）
- 政策解读材料（官方解读、媒体解读、专家解读）
- 政策影响和舆论反应

**搜索次数要求**：≥54次

## 政策维度评分标准

| 评分 | 标准 | 来源类型 |
|------|------|---------|
| **5分** | 政府官网政策原文（含文号、发布机构、核心条款） | gov.cn/nea.gov.cn/miit.gov.cn等 |
| **4分** | 权威媒体政策报道（新华社/人民日报/经济日报），含政策要点 | xinhuanet.com/people.com.cn/ce.cn |
| **3分** | 行业媒体政策解读或一般媒体政策报道 | 行业媒体/综合媒体 |
| **2分** | 内容泛泛、缺乏政策具体条款，或非当月政策 | 自媒体/论坛 |
| **1分** | 无关内容 | 广告 |

## 政策维度"充分"标准

| 检查项 | 充分标准 |
|--------|---------|
| 国家政策 | ≥3条（来自≥2个不同部委） |
| 地方政策 | ≥5条（来自≥5个不同省份） |
| 政策解读 | ≥3条（官方解读+媒体解读） |
| 政策影响 | ≥2条 |
| 来源多样性 | 至少有5条来自gov.cn域名 |

## 来源偏好规则（政策维度专用）

```
⚠️ 政策维度的URL必须优先使用政府来源：

第1优先：政府部门官网
  - gov.cn（国务院）
  - miit.gov.cn（工信部）
  - ndrc.gov.cn（发改委）
  - mof.gov.cn（财政部）
  - nea.gov.cn（国家能源局）
  - 地方政府官网（如fgw.nmg.gov.cn）

第2优先：权威中文媒体
  - xinhuanet.com（新华社）
  - people.com.cn（人民日报）
  - ce.cn（经济日报）
  - jjckb.cn（经济参考报）

第3优先：行业专业媒体
  - in-en.com
  - china-nengyuan.com

禁止：
  - 英文媒体URL
  - 企业官网URL
  - 企业首页/政府首页（如 https://www.sinopec.com.cn/）
```

## 弹性搜索（按需触发）

> **触发条件**：当基础搜索（P0-P50）完成后，覆盖度评估显示"不足"时，按以下优先级补充搜索。每轮补充1-3次，最多3轮。连续3轮无新增≥3分结果则标记"已搜尽"。

### 补充搜索优先级

| 优先级 | 补充方向 | 搜索语句模板 |
|--------|---------|-------------|
| 1 | 按热点深挖 | `"[热点关键词]" (配套 OR 细则 OR 影响) [月份] [年份]` |
| 2 | 按未覆盖省份补充 | `"氢能" site:[未搜索省份发改委域名] [月份] [年份]` |
| 3 | 按权威媒体补充 | `"氢能" (政策 OR 试点 OR 示范) site:[未搜索媒体域名] [月份] [年份]` |
| 4 | 换词重搜 | 将"政策"→"规划"/"方案"/"指导意见"等同义词替换后重搜 |
| 5 | 按政策类型补充 | `"(零碳工厂 OR 绿色工厂 OR 氢能标准)" [月份] [年份]` |

### 换词原则

- 更换同义词：`政策`→`规划`→`方案`→`指导意见`→`通知`
- 更换区域：`内蒙古`→`新疆`→`甘肃`→`宁夏`→`青海`
- 更换来源：`site:xinhuanet.com`→`site:people.com.cn`→`site:ce.cn`
